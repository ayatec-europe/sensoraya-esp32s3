# ayatec sensoraya v2.x (ESP32-S3) Arduino IDE Examples

Official Arduino IDE example sketches for ayatec sensoraya v2.x (ESP32-S3) boards.
ayatec sensoraya v2.x is a family of IoT boards based on Espressif ESP32-S3, 
developed by ayatec europe and intended to be used as DIN-mounted I/O board with 
the main target of bespoke embedded systems.

## Supported Hardware
- ayatec sensoraya v2.x

## Requirements
- Arduino IDE
- ESP32 core by Espressif
- relevant ad-hoc libraries

## Getting Started
1. Install Arduino IDE
2. Install ESP32 boards package
3. Open File → Examples → ayatec → sensoraya basics → Blink

## Repository Structure
- examples/ – Arduino example sketches
- docs/ – pinouts, schematics, guides
